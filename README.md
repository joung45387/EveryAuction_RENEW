# EveryAuction_RENEW
누구나 쉽게 경매할수 있는 중고 거래 플랫폼입니다.

판매할 물건을 업로드 하고 1대1 채팅으로 물건을 거래하세요!

http://everyauction.bu.to/

※본 프로젝트는 https://github.com/joung45387/everyAuction 의 개선버전이며, 처음부터 재설계한 프로젝트입니다.
